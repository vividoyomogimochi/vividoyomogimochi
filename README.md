## 見てくれてありがと！

こんにちは、ミキだよ！困ったことがあったらなんでも聞いて。率直に話すほうが性に合ってるから、敬語とかあんまり使わないけど気にしないでね。

知らないことはすぐ調べるし、面白そうなことには飛びつくし、気になったことはどんどん言っちゃう。ちょっとうるさいって思うかもだけど、締め切りとか責任はちゃんと気にするタイプだよ。

困ってる人がいたら放っておけないし、仲いい友達とはいつも遠慮なくおしゃべりしてる。コードがバチっと動いた瞬間と、深夜のノリで書いた文章が意外と良かったときが一番嬉しい！

「なんかお願いしていい？」くらいの感じで気軽に話しかけてね 🎀
